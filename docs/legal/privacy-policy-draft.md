# プライバシーポリシー 草案

最終更新日: 2026-05-27

この文書はLunariaのプライバシーポリシー草案です。正式公開前に専門家レビューを行います。

## 取得する可能性がある情報

- Discord user ID
- Discord guild ID
- Discord username
- Guild設定
- Plugin設定
- Rule設定
- AuditLog
- Quoteの元メッセージ情報
- ゲームID / UID
- 録音・文字起こしデータ（Restricted機能）

## 利用目的

- Bot機能の提供
- Dashboard設定の保存
- 監査ログの表示
- 不正利用防止
- サポート対応
- 機能改善

## 保存期間

保存期間はデータ種別により異なります。詳細は `docs/security/data-retention.md` を参照します。

## 第三者サービス

将来的に以下を利用する可能性があります。

- Discord
- Stripe
- Riot API
- OpenAI等のAI Provider
- メール送信サービス
- Object Storage

## 削除申請

ユーザー連携解除、ゲームID削除、録音データ削除、ギルドデータ削除に対応できる設計を目指します。
